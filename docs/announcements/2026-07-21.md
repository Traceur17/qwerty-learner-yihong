## 致 小圆饼干

小圆饼干，你好！这次主要打磨了**连播卷面底栏与操作体验**，对答案后也能看到正确率。

### 连播 · 底栏更像指标栏

底栏改为与正常听写指标栏相近的半透明分栏：播放 / 间隔 / 计时 / 对答案；对答案后正确率会平滑展开。

- 播放、对答案、收起用主题色图标，一眼可点
- 对答案后显示正确率（已判分行答对占比）
- 收起答案时正确率栏收回

### 连播 · 操作小改进

- `←` / `→` 只移动输入框内光标；上下切词仍用 Enter / Tab / ↑↓
- 对答案后，点击词条卡片（题号除外）可单独发音
- 题号区加宽，三位数 + 播放符不易挤进输入框
- 滚动条默认半透明，悬停变实，少挡右下角装饰
