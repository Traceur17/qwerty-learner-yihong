## 致 小圆饼干

小圆饼干，你好！这次连播模式新增了**正确率展示**，并做了一轮 **UI/UX 优化**。

### 连播 · 正确率展示

对答案后，底栏会平滑展开正确率（已判分行中答对占比）；收起答案时收回。底栏整体也更接近正常听写的指标栏样式。

### 连播 · UI/UX 优化

- **左右快捷键**：`←` / `→` 只移动输入框内光标，不再上下切词（切词仍用 Enter / Tab / ↑↓）
- **滚动条遮挡**：滚动条默认半透明，悬停再变实，减轻挡住右下角装饰的问题
- **点击卡片发音**：对答案后，点击词条卡片（题号除外）可单独听发音
- **题号加宽**：三位数 + 播放符不易挤进后面的输入框
- **底栏按钮**：播放 / 暂停、对答案 / 收起改为主题色图标，更容易辨认可点击
