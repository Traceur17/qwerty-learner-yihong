# 连播卷面底栏与操作体验

日期：2026-07-21

## 背景

连播底栏原先偏工具条样式，与正常听写指标栏不一致；左右方向键占用输入框光标移动；对答案后缺少整体正确率与便捷复听。

## 范围

1. **底栏指标化**：半透明卡片 + 分栏；顺序为播放 / 间隔 / 计时 / 对答案；对答案后在计时与操作之间展开正确率，收起时收回。正确率 = 已判分行中答对占比。播放/对答案/收起用主题色图标按钮。
2. **键盘**：`←`/`→` 仅移动光标；`Enter`/`Tab`/`↑↓` 切行（既有焦点上限不变）。
3. **对答案后点卡片发音**：已判分行点击卡片区域（题号除外）调用 `playChapterWord`。
4. **题号列加宽**；**滚动条**默认半透明、悬停变实。

## 非目标

- 不进入章节结算页；正确率仅展示于连播底栏。
- 不改判分写库与错答历史逻辑。

## 实现要点

- `ContinuousDictationSheet` 底栏；正确率用 `max-width` + opacity 过渡。
- `openspec/specs/continuous-dictation-sheet/spec.md` 已同步左右键行为。
