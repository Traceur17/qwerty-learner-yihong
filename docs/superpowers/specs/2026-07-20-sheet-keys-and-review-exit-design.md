# 连播卷面键盘增强 + 错题练习结算回原章

日期：2026-07-20

## 背景

连播卷面输入焦点过严（只能跟到当前播放词），切词缺少左右方向键，暂停快捷键 Esc 太远。  
章节结算进入的「本章错词练习」结束后退出时被强制重置到第 1 章，缺少回到原章继续练的入口。

## 范围

### A. 连播卷面键盘（仅 ContinuousDictationSheet）

1. **焦点上限**：由「当前播放指针」改为「当前播放指针 + 1」（不超过词表末尾）。例：播到第 10 词时，最多可聚焦第 11 词；再往后仍不可聚焦。
2. **左右切词**：输入框内 `←` 上一词、`→` 下一词；仍受焦点上限约束。保留现有 `Enter` / `Tab` / `↓` 下一词，`Shift+Tab` / `↑` 上一词。
3. **暂停/继续**：`Ctrl + Space` 同一快捷键切换播放/暂停；输入框内外均生效（`preventDefault`）。保留：非输入焦点时单独 Space 切换；Esc 现有行为不变（关钉住历史 / 暂停 / 收起揭晓）。

### B. 错题练习结算回原章（ResultScreen，有原章快照时）

适用：结算页「再练本章错词」、训练页错题本等带 `chapterErrorReturn` 的本章错词练习。  
不适用：Gallery 全局错题复习（无原章快照）——仍保留「练习其他章节」→ 词库页。

有 `chapterErrorReturn` 时：

1. **退出复习不再重置到第 1 章**，定位到快照中的原章节。
2. 结算页按钮改为：
   - **重新练习本章节**：退出复习 → 回到原章词表 → 重复本章（等同正常结算的「重复本章节」）。
   - **下一章节**：退出复习 → 原章的下一章（末章则不展示或禁用，与正常结算一致）。
3. **关闭（×）**：等同「重新练习本章节」。
4. 可保留次要入口「练习其他章节」→ 退出复习并跳词库（不强制；若保留则不再把章节置 0，可保持原章或按现有词库入口习惯）。

推荐默认按钮文案与正常章节结算对齐：`重复本章节`、`下一章节`；关窗 = 重复本章节。

## 非目标

- 不改错题「最新状态」口径、卷面判分写库、天分激励阈值。
- 不改 Gallery 无快照错题复习的结算主流程（仍可走词库）。
- 不改连播卷面判分、错答历史 UI。

## 实现要点

- `focusRow`：`Math.min(index, playIndex + 1, words.length - 1)`。
- 全局 keydown：识别 `e.ctrlKey && (e.key === ' ' || e.code === 'Space')` 切换播放。
- `onAnswerKeyDown`：增加 `ArrowLeft` / `ArrowRight`。
- ResultScreen：读 `reviewModeInfo.chapterErrorReturn`；退出 helper 统一「清复习态 + 设回原章 + REPEAT_CHAPTER / NEXT_CHAPTER」。

## 验收

- 连播：播到 10 可进 11 输入；不可进 12；左右键切词；Ctrl+Space 输入中可暂停/继续。
- 本章错词练完结算：× / 重复本章节 → 原章重练；下一章节 → 原章+1；不再落到第 1 章。
