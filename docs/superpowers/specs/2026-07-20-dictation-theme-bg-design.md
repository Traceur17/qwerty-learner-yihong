# 听写主题装饰背景（荷花）

日期：2026-07-20

## 背景

听写模式希望右下角有一抹淡淡诗意装饰；当前主题为荷花（饼干养荷花）。后续可能换其他主题图。

## 设计

- **显示条件**：`listenDictationConfig.isOpen`（含逐词听写与连播卷面）
- **位置**：练习主区域右下角，`pointer-events-none`，不挡交互
- **样式**：透明底 PNG；约 12%–18% 不透明度；宽度约 `min(28vw, 360px)`
- **资源**：`src/assets/dictation-themes/`；主题表登记 id / 图 / 名称；默认 `lotus`
- **本期**：只做默认荷花主题，不做主题选择 UI

## 素材处理

源图带白底 → 近白像素转透明，导出干净 PNG，便于融入页面底色与暗色模式。
