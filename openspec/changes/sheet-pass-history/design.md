## Context

连播卷面（`ContinuousDictationSheet`）已支持连播、对答案揭晓、词级 `wordRecord` / `wrongAnswerHistories`，但没有「整章第 N 遍」快照。探索结论：饼干节奏多为整章一遍；需要折线进步信号 + Excel 式多遍对比满足感；存储压力可忽略。

约束：中途「对答案」必须保持可用；同轮改字再对不能打出折线毛刺；打开默认恢复草稿，不弹「加载 or 从头」选择。

## Goals / Non-Goals

**Goals:**

- 按 `(dict, chapter)` 持久化当前草稿，打开默认恢复。
- 可封存条件（已播完整章 + 已对过答案）满足时：再练一遍 / 切章或离开 → 写入完整历史（同轮 upsert）。
- 底栏正确率 → 进步弹窗（折线）→「查看听写记录」全页宽表（行=题、列=遍），右上角关闭。
- 「再练一遍」区分可封存与不可封存（警告清空）。

**Non-Goals:**

- 不改错题本最新状态口径；不替代 `wrongAnswerHistories` 圆点。
- 不强制一词一判模式的遍次历史。
- 不做全站 Analysis 级全局听写趋势（本章维度即可）。
- 首版不做「仅错题行」过滤、密度自动降级、移动端专用双列对比（可后续加）。

## Decisions

### D1 · 数据模型

- **决定**：Dexie 增加：
  - `sheetPassDrafts`：主键或唯一键 `(dict, chapter)`，存 `answers[]`、`grades[]`、`playIndex` / `maxPlayedIndex`、`revealed`、`updatedAt`，以及可选 `linkedPassId`（若已对应某条历史）。
  - `sheetPasses`：完整历史；字段含 `dict`、`chapter`、`timeStamp`、`accuracy`、`correctCount`、`wrongCount`、`totalGraded`、`answers: string[]`、`grades: ('correct'|'wrong')[]`、`wordNames: string[]`（防词表变更对不齐时的锚点）。
- **备选**：只扩展 `wordRecords` —— 无法表达整卷列对比与 session 边界。
- **理由**：文本快照体积极小；与错题体系解耦。

### D2 · 何时写入完整历史（封存）

- **可封存**：`maxPlayedIndex === lastWordIndex` 且本轮至少成功对答案一次（`revealed` 且已播范围均有 grade）。
- **写入时机**：
  1. 「再练一遍」且可封存 → upsert 后清空草稿开新轮；
  2. 切章 / 卸载卷面组件（离开练习）且可封存 → 静默 upsert（等同手动保存）。
- **同轮 upsert**：草稿带 `linkedPassId` 时更新该条；否则新建并回写 id。多次「对答案」只更新草稿与（若已封存过）同一历史行，不新增遍次。
- **中途对答案**：只更新草稿 + 现有词级记录；**不**新建 `sheetPasses` 行。

### D3 · 默认加载草稿

- 进入该章卷面时读取 draft；有则恢复答案/播放进度/揭晓态；无则空卷。
- 不弹窗询问。重来唯一显式出口：「再练一遍」。

### D4 · 「再练一遍」

- 可封存：先 upsert `sheetPasses`，再清空草稿与卷面状态。
- 不可封存：确认对话框「将清空本轮记录，且不记入完整历史」→ 确认后清空；取消则保持。

### D5 · UI 信息架构

```
底栏正确率（可点，有历史或当前揭晓时）
  → 进步 Dialog（中等尺寸）：折线 + 摘要 + [查看听写记录]
       → 全页 Dialog/面板：Excel 宽表（可含顶部迷你折线）
            右上角 ✕ 关闭回卷面（或先关全页留进步窗，首版：关全页即回到卷面）
```

- 折线横轴 = 第 N 遍；点 hover 显示日期与对/错题数；点击某点 → 宽表滚到并高亮该列。
- 宽表：冻结题号 + 正确答案列；每列一遍（表头：第 N 遍、日期、正确率）；格内用户所写 + 绿/红；点错格可看 diff。
- 可选最右「本轮」列展示未封存草稿（进行中样式）；首版若实现成本高可省略，以卷面本身为准。

### D6 · 正确率入口可见性

- 当前轮已揭晓：显示本轮正确率并可点。
- 未揭晓但该章已有 `sheetPasses`：仍显示可点入口（可用最近一遍正确率或文案「历史」），避免只能对完答案才能回顾。
- 无历史且未揭晓：入口弱化或隐藏。

### D7 · 折线实现

- 复用 Analysis `LineCharts` 的可视化思路或轻量 SVG/现有图表依赖；数据源仅为本章 `sheetPasses` 按时间序。
- 0 遍：空态文案；1 遍：单点 +「再完整听写一次可看趋势」。

## Risks / Trade-offs

- [词表/章节切分变更导致历史列与当前 words 错位] → 存 `wordNames`；展示按历史快照行渲染，与当前词表 merge 时按 name 对齐，对不上的行灰显。
- [离开时静默写入与「再练」重复写] → 统一 upsert by `linkedPassId` / 草稿指纹。
- [全页宽表列很多] → 横向滚动 + 默认滚到最新列；首版不做虚拟列。
- [草稿恢复后像「已经考完」] → 预期满足感；「再练一遍」出口明确即可。

## Migration Plan

1. Dexie version bump：新表默认空。
2. 无旧数据迁移；错史/wordRecord 不变。
3. 回滚：隐藏 UI 入口；表可保留无害。

## Open Questions

- 全页关闭后是否保留进步弹窗：首版关闭全页直接回卷面（更简单）。
- 「本轮」列是否首版就做：倾向不做，降低范围。
