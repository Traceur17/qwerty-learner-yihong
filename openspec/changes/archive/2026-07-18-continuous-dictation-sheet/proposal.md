## Why

现有听写是「一词一播、即时判对错」，难以训练雅思机考听力所需的技能：边听连续音频、在编号答题格间赶题、听完（或中途）再对账。长章（100 ～ 400 词）词表需要「卷面式」连播练习，且必须用播放指针防止整章错位导致对账失真。

## What Changes

- 新增 **连播听写卷面**模式：纵向编号表 + 作答格，按可配置间隔自动连播词音频
- 播放控制：暂停 / 继续；点击题号从该题重新起播；行高亮表示当前播放位置
- 焦点导航：`Enter` / `Tab` / `↓` 下一格，`Shift+Tab` / `↑` 上一格；输入焦点不得超过当前播放题号
- 默认识别列隐藏：仅显示题号与作答；点击「对答案」后才展示英文、释义、错史，且只对**已播放**题判分
- 对账 UI：正确有明显 ✓；错误显示直观 diff
- 错史：最近一次完整错答（带 diff）+ `×N`；点击展开近 3 次；无历史显示 `/`；落库保留最多近 3 次错答字符串
- 入口：与现有一词一判听写并存（设置或模式开关）；不替换现有听写反馈流程
- 复用现有词级裁切 MP3 与发音倍速配置；不引入整章原音合并

## Capabilities

### New Capabilities

- `continuous-dictation-sheet`: 连播听写卷面的播放、导航、对账、错史展示与持久化

### Modified Capabilities

- （无）现有 `listen-dictation-mode` 一词一判行为保持不变；本能力为并列模式

## Impact

- UI：`src/pages/Typing/` 新增卷面组件；设置/工具栏增加模式与间隔配置
- 状态：Jotai 配置 atom（模式开关、连播间隔）；卷面 session 状态（播放指针、作答数组、是否已对账）
- 音频：复用 `usePronunciationSound` / `wordAudioPlayer`；定时连播调度
- 数据：`wordRecords` 或扩展字段保存近 3 次错答字符串；错题本可展示错史
- 与 `randomConfig`、现有听写 diff 工具可复用部分逻辑
